تقرير تحليل البيانات للتطبيق
1. تحميل ومعالجة البيانات

    • تم تحميل بيانات المستخدمين والإيرادات.
    • تم فحص البيانات وتنظيفها، بما في ذلك التعامل مع القيم المفقودة (NaN) واستبدالها بالقيم المناسبة.
    • تم تحويل الأعمدة التي تحتوي على تواريخ إلى نوع البيانات الصحيح (datetime).
    • تم دمج بيانات المستخدمين والإيرادات باستخدام `user_id`.
    
2. تحليل الإيرادات

    • تم حساب الإيرادات الصافية (`net_revenue`) والتي تساوي (الإيرادات - المبالغ المستردة).
    • تم حساب الإيرادات حسب نوع الإعلان باستخدام `groupby` و`sum`.
    • تم حساب عدد المعاملات لكل نوع إعلان.
    
3. معدل الاحتفاظ

    • تم حساب معدل الاحتفاظ في الأيام 1، 3، و7 من تثبيت التطبيق باستخدام الفارق بين تاريخ التثبيت وتاريخ المعاملة.
    
4. اختبار ANOVA

    • تم استخدام اختبار ANOVA للتحقق من وجود فروقات ذات دلالة إحصائية بين المجموعات (الاختلاف في ترتيب عرض الألعاب).
    • نتائج الاختبار:
    • F-statistic: 0.36
    • p-value: 0.78
    • الاستنتاج: بما أن p-value > 0.05، لا يوجد فرق ذو دلالة إحصائية بين المجموعات.
    
5. تحليل معدل التحويل

    • تم حساب معدل التحويل للمستخدمين الذين قاموا بعملية Cashout وكذلك المعاملات الأخرى.
    • تم حساب الإيرادات الإجمالية وعرض الإيرادات المتوسطة لكل عملية تحويل.
    
6. الاستكشاف بالرسوم البيانية

    • رسم توزيع الإيرادات باستخدام **Histogram**.
    • رسم توزيع المعاملات حسب نوع الإعلان باستخدام **countplot**.
    • رسم الإيرادات حسب نوع الإعلان باستخدام **boxplot**.
    • عرض الإيرادات حسب البلد باستخدام **barplot**.

التوصيات

    1. **تحسين الإيرادات من الإعلانات الأكثر فعالية:**
       تعزيز إعلانات Interstitial، ومراجعة وتحليل سبب انخفاض إعلانات Unknown.

    2. **زيادة معدل الاحتفاظ:**
       تقديم محتوى مميز أو مكافآت إضافية في الأيام التالية للتثبيت، واستخدام إشعارات تفاعلية.

    3. **مراجعة استراتيجية ترتيب عرض الألعاب:**
       إعادة النظر في عرض الألعاب أو البحث عن عوامل أخرى تؤثر على الإيرادات.

    4. **زيادة معدل التحويل:**
       تحسين عملية Cashout لتكون واضحة وسهلة، مع توفير تحفيزات إضافية.

    5. **تحليل الإيرادات حسب البلد:**
       التركيز على الدول الأعلى تحقيقًا للإيرادات، مثل الولايات المتحدة، وتحسين الاستهداف الجغرافي للإعلانات.

    6. **التوسع في تحليل البيانات:**
       دراسة أوقات الذروة للإستخدام، والمكافآت، والعروض الترويجية لتحفيز الاستخدام المتكرر.
